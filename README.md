## Freelance Frontend
a modern app built with react and material UI, designed as part of a full stack freelance platform project.
##  Features
- Authentication ready (JWT integration ready)
- API integration using Axios and React query
- Fully responsive design (mobile & desktop)
- Clean and scalable UI structure
- Component-based architecture
##  Tech Stack
- react.js
- MUI
- Axios
- react router dom
- react query
- javaScript 
- vite
